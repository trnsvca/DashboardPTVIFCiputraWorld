# PT Victory Analytics Dashboard

Interactive dashboard for analyzing XAUUSD pivot points (2017-2025) and customer data (H1 2026).

## Features
- **Cover Page**: Overview of cumulative growth.
- **Pivot XAUUSD Page**: Detailed analysis of trading performance with date filtering.
- **Data Nasabah Page**: Demographic and transaction statistics for 2026.
- **Interactive Charts**: Built with Plotly for deep exploration.

## How to Run Locally
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## Deployment
This app is ready to be deployed on **Streamlit Cloud**:
1. Push this code to a GitHub repository.
2. Go to [share.streamlit.io](https://share.streamlit.io).
3. Connect your GitHub account and select this repository.
4. Click **Deploy**.

## Dataset
The data is sourced from a Google Spreadsheet, exported as an Excel file.
